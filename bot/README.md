# Елена Насырова Бот

Telegram-бот помощник Елены Насыровой. Проверяет подписку на канал и после подписки выдает материалы: сакрамедитации, ссылку на спектакль-практику, Rutube и соцсети.

## Как это работает

1. Человек нажимает кнопку на сайте.
2. Открывается Telegram-бот с параметром, например:
   `https://t.me/ElenaNasyrovaBot?start=sacrameditations`
3. Бот просит подписаться на канал.
4. Человек подписывается и нажимает `Я подписался`.
5. Бот проверяет подписку через Telegram API.
6. Если подписка есть, бот отдает материалы и дополнительные ссылки.

## Что нужно от Telegram

1. Создать бота через `@BotFather`.
2. Скопировать токен в `bot/.env`.
3. Добавить бота администратором в канал `Кластер уникальности`.
4. Узнать `CHANNEL_ID`.

Для приватного канала нужен numeric id вида `-1001234567890`. Invite-ссылка вида `https://t.me/+...` нужна только для кнопки подписки, по ней бот подписку не проверяет.

## Запуск локально

```bash
cd bot
copy .env.example .env
npm start
```

В `.env` нужно заменить:

- `BOT_TOKEN`
- `CHANNEL_ID`
- `SACRAMEDITATIONS_URL`, когда будет ссылка на аудиоматериалы
- `SILA_RODA_RUTUBE_URL`, когда будет ссылка на спектакль

## Команды бота

- `/start` — стартовое меню
- `/sacrameditations` — получить сакрамедитации после проверки подписки
- `/sila_roda` — получить спектакль-практику после проверки подписки
- `/socials` — соцсети и контакты
- `/help` — помощь

## Ссылки для сайта

Когда будет username бота, ссылки на сайте лучше делать такими:

```text
https://t.me/BOT_USERNAME?start=sacrameditations
https://t.me/BOT_USERNAME?start=sila_roda
```

`BOT_USERNAME` заменить на имя, которое выдаст BotFather.
